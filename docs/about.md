# ChoTech（旧 TechHive）について

**共に学び、共に創り、共に発信する。情報系学生のための新しい活動拠点**

---

## ChoTech とは？

ChoTech は、**長崎大学 情報データ科学部**の学生（学部生・院生）が主役となり、技術を軸に集い、学びを深め、新たな価値を創造していくためのコミュニティです。

私たちは、皆さんが主体的かつ協働的に **「技術を学ぶ・作る・話す」** 独自のカルチャーを育んでいくことを目指しています。

---

## 私たちの想い

急速に進化する情報技術の世界では、実践的なスキルとアウトプットの機会が不可欠です。しかし、自主学習だけではモチベーションの維持が難しかったり、質の高い学びの機会に金銭的な制約を感じたりすることも少なくありません。

私たちは、こうした「学びの壁」を感じている学生が、**仲間と共にその壁を乗り越え、技術探求の楽しさを見いだせる場**を創りたいと考えています。

GitHub や Zenn、Qiita などの技術系プラットフォームの活用ノウハウや、先人たちが培ってきた知識・経験を共有し合うことで、誰もが実践的なスキルを磨き、発信できる環境を提供します。

---

## 主な活動内容

ChoTech は、学生の皆さんの **「インプットとアウトプットの場」** を提供し、幅広い知識と専門性を兼ね備えた **「T 字型人材」** の育成に貢献します。

### 🎤 Talk Day（ライトニングトーク・アイデアプレゼン）

| 項目       | 内容                                                                 |
| ---------- | -------------------------------------------------------------------- |
| **開催頻度** | 月 1〜2 回（目安）                                                    |
| **形式**   | ライトニングトーク（LT）— 1 人 5 分程度のショートプレゼン              |
| **内容**   | IT 技術に関連するカジュアルなプレゼンテーションやアイデアソンを実施 |

#### 目的

- 初心者でも気軽に登壇できる環境で、学会発表などに向けた**プレゼンテーション能力**を養う
- 多様な分野の知識（リベラルアーツ）に触れ、自身の興味や探究心を刺激するきっかけを提供
- アイデアを形にし、他者と共有する楽しさを体験

---

### 💻 Dev Day（技術勉強会・ハンズオン）

| 項目       | 内容                                                                       |
| ---------- | -------------------------------------------------------------------------- |
| **開催頻度** | 月 1〜2 回（目安）                                                          |
| **形式**   | ゼミ形式のディスカッション、ハンズオン形式など多様なスタイル               |
| **内容**   | 講義で得た知識の深掘り、実社会で役立つ応用技術・開発手法の探求             |

#### 目的

- 座学だけでは得られない、より**実践的で深い学び**の機会を提供
- 参加者同士で知識やスキルを教え合い、共に技術力を高める

---

## コミュニティが目指すもの

私たちは「情報データ科学」を共通言語として、単に知識やスキルを習得するだけでなく、以下の価値をコミュニティ全体で追求していきます。

| 価値                         | 説明                                                                           |
| ---------------------------- | ------------------------------------------------------------------------------ |
| 💬 **コミュニケーションの大切さ** | 一人で勉強するだけでなく、みんなで話し合い、助け合いながら問題解決する力を身につける |
| 🤝 **お互いに高め合う環境**       | 仲間と一緒に学ぶことで、新しい技術やアイデアに挑戦する意欲が生まれる               |
| ✨ **学びの喜びを共有**           | 新しい発見を通じて学ぶ楽しさを感じ、その経験を仲間と共有する                       |

---
